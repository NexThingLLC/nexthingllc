## Project Overview

A simple business web site that will be fontend code only and capable of being hosted on github pages.  The site is for NexThing, LLC. a consulting company offering consulting services in Cloud, Automation, Site Reliability Engineering, DevOps and Agentic AI.

## Site Style

The site style will be business professional, yet modern and edgy.  Use logo.png for the site logo and the colors of the site should compliment the colors in the logo.

## Sections

The site should contain sections for about us, our promise, our expertise and contact us.  The about us section should include a quote from Alan Turing - "We can only see a short distance ahead, but we can see plenty there that needs to be done."
